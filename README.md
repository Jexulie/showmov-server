# ShowMov -  Back-end Side

Tiny Server for ShowMov-App.
 
Shamelessly scrapped from a certain site with web-scapper.

### Usage

#### Movies:
---

> 'GET' Fetching Movies for Database.

``` 
    /api/v1/movies/fetch
```

> 'GET' Fetching Movies.

``` 
    /api/v1/movies/get
```

> 'DELETE' Clearing Movie Database Collection.

``` 
    /api/v1/movies/clear
```
