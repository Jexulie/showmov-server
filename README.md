# ShowMov -  Back-end Side

## Usage

> Fetching Movies for Database.

``` 
    /api/v1/fetchmovies
```

> Clearing Database Collection.

``` 
    /api/v1/clearall
```

> Fetching Movies.

``` 
    /api/v1/getmovies
```
